## Pre-Requisities-
vercel
python version<=3.10
IPFS server
node.js

## Instructions

## Install-
Step 1- 
```
npm install
```

Step 2- 
```
  cd ml-backend
  pip3 install -r requirement.txt
```

Step3- 
  Setup IPFS server for PC (recommended)

### Run- 
## Locally-
```nodemon app.js```
